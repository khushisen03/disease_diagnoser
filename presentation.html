Matching Algorithm

  1. Symptom Collection: Gathers all user symptoms
  2. Normalization: Converts to lowercase
  3. Pattern Matching: Finds overlapping symptoms
  4. Score Calculation: Match Score = (Matching Symptoms) / (Max Symptoms)
  5. Ranking: Sorts by match score
  6. Filtering: Shows top 5 results

Disease Database

  • Common Cold: fever, cough, headache → Acetaminophen, Ibuprofen
  • Pneumonia: fever, cough, chest pain → Amoxicillin, Azithromycin
  • Migraine: headache, nausea, fatigue → Sumatriptan, Ibuprofen
  • Flu: fever, sweating, fatigue → Oseltamivir, Acetaminophen
  • Heart Disease: chest pain, shortness of breath → Aspirin, Atorvastatin
